1. Fetch F1 race data from the Jolpica API (2022–2024) — requires internet connection.

## Features Engineered

| Feature | Type | Description |
|---------|------|-------------|
| `driver_prev_grid` | Lag | Driver's grid position in previous race |
| `driver_rolling_top10_rate` | Rolling aggregate | Driver's top-10 rate over last 3 races |
| `driver_circuit_top10_rate` | Interaction / historical | Driver's top-10 rate at this specific circuit (expanding) |
| `constructor_rolling_top10_rate` | Rolling aggregate (team) | Team's top-10 entry rate over last 5 races |
| `constructor_tier` | Categorical encoding | Team tier (1=Top, 2=Mid, 3=Back) from training-set average points |

All features pass the leakage guard checklist (timing ✓, `.shift(1)` computation ✓, target isolation ✓).
